安装/重装Mysql必备步骤

下载相关软件

下载对应MySQL版本

Download MySQL Community Server

下载对应MySQL Workbench版本

Download MySQL Workbench

下载完后解压出来的deb文件可以直接右键用新立得软件中心图形安装
不过要注意安装的依赖顺序
注：如果安装mysql-community-client_5.7.12-1ubuntu15.10_amd64.deb需要依赖libbaio1的安装包，我个人的解决办法是更新源（阿里源），然后update, upgrade 就会自动装上，如果与之前安装的有冲突，直接去路径中删除文件，再次安装就可以成功了
